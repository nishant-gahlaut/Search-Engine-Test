b'\nA permutation $P$ of length $N$ ($1$-based) is called good if it satisfies the following these conditions:\n\nFor each $1 \\le i \\le N$, $P_i \\neq i$\n$|P_1 - 1| \\oplus |P_2 - 2| \\oplus \\ldots \\oplus |P_N - N| = 0$\n\nHere, $\\oplus$ denotes the bitwise XOR operation and $|X|$ denotes the absolute value of $X$.\nFind any good permutation $P$ of length $N$ or print $-1$ if no such permutation exists.\nNote: A permutation of length $N$ is an array which contains every integer from $1$ to $N$ (inclusive) exactly once.\nInput Format\n\nThe first line contains a single integer $T$ - the number of test cases. Then the test cases follow.\nThe first and only line of each test case contains an integer $N$ - the length of the permutation $P$.\n\nOutput Format\nFor each test case, print $-1$ if no such permutation exists. Otherwise, in a single line, output $N$ space-separated integers - the required good permutation. If more than one solution exists, you may print any of them.\nConstraints\n\n$1 \\leq T \\leq 10^6$\n$1 \\leq N \\leq 10^6$\nSum of $N$ over all test cases does not exceed $10^6$\n\n\n                        Sample Input 1\n                        \n\n\n\n\n\n3\n1\n2\n8\n\n\n\n                        Sample Output 1\n                        \n\n\n\n\n\n-1\n2 1\n7 1 5 6 3 4 8 2\n\n\nExplanation\nTest case 1: There is only one permutation $P = [1]$ of length $1$. Since $P_1 = 1$ the answer does not exist.\nTest case 2: Consider the permutation $P = [2, 1]$.
